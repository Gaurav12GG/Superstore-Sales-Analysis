# Superstore-Sales-Analysis
# 📊 Superstore Sales Analysis (Python + SQL)

## 📌 Project Overview

This project analyzes a Superstore sales dataset to uncover insights related to customer behavior, product performance, and regional sales trends.

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* SQL (SQLite)
* Jupyter Notebook

## 📂 Dataset

* Superstore dataset containing ~9800 records
* Includes sales, customer, product, and region details

## 🔍 Key Analysis Performed

* Data Cleaning (missing values, duplicates, date conversion)
* Sales Analysis by:

  * Region
  * Category
  * Segment
  * State (Top 10)
  * Top Products
* Monthly Sales Trend Analysis
* SQL Queries for data extraction and insights

## 📊 Key Insights

* Technology category generates highest revenue
* Consumer segment dominates sales
* West region has highest sales performance
* Sales peak during festive season (Nov-Dec)
* Few top products contribute major revenue

## 🎯 Conclusion

The business shows strong growth and seasonal trends. Focusing on high-performing products, key regions, and festive demand can significantly improve revenue.
